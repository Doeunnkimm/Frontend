## Jest 및 테스팅 라이브러리로 React 테스트하기

### jest-dom으로 처음 테스트해보기

```bash
$ npm test
```

- create-react-app과 함께 설치되는 jest-dom 실행
- setupTest.js파일을 사용해 각 테스트 전에 jest-dom을 가져온다.
- 즉, 모든 테스트에서 jest-dom 매처를 사용할 수 있다.
