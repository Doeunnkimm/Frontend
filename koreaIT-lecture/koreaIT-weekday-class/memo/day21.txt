자바스크립트(JavaScript)
	정적인 HTML 웹 페이지의 제한적인 기능을 동적(click, edit)으로 변화시키는 엡 개발 언어이다.
	자바스크립트로 작성된 문서는 클라이언트에 그대로 전달되어 클라이언트에서 동작한다.

특징
	자바스크립트는 인터프리터 언어로 컴파일 언어(자바, C 등)가 아니다. (01: 35: ..)
	인터프리터 언어는 기본적으로 실행 직전에 코드를 읽어 바로바로 번역하여 동작한다.
	입력된 코드가 클라이언트에 그대로 전송되므로 실행시 클라이언트의 자원을 사용한다.
	자바스크립트 코드는 기본적으로 HTML 문서에 직접 코딩하여 사용한다.
	자바 또는 C언어와 다르게 변수 타입을 따로 정의하지 않는다.(모든 변수는 var)
	자바/C int, double, boolean, char, String ...

HTML		웹 페이지의 뼈대에 해당하는 웹 페이지의 구조를 정의한다.
CSS		웹 페이지의 외형에 해당하는 레이아웃 및 디자인을 변경한다.
JavaScript	웹 페이지의 두뇌에 해당하는 동적 기능을 추가하여 동작하도록 한다.

ECMA(기술 및 통신업계 국제적 비영리 표준기관)
	CCMAScript6 - ES6, ECMA 기술 규격에 정의된 표준화된 스크립트 프로그램밍 언어

PS	
	자바스크립트는 원래 웹페이지의 동적인 기능을 프로그래밍하기 위한 언어로 출발하였으나
	점점 그 성능과 기능의 발전 및 확장되어 현재는 클라이언트 뿐만 아니라 서버 그리고
	PC 및 모바일 기기등에서 컨버팅하여 동작할 수 있도록 영역이 확장되고 있다.
	그 영역 확장의 중심에 대표적으로 Node.js가 있다.

자바스크립트 대표적 작성 방법

내부(Internal)
	HTML 문서 내에서 영역을 구분해서 기술한다.

	<script>
		document.write("Hello World!");
	</script>

외부(External)
	외부 파일에 자바스크립트만 작성하여 파일을 불러와 사용한다.

	<script scr="./welcome.js">
		
	</script>

인라인(Inline)
	태그 안에 직접 기입하는 방식이다.

	<input type="button" onclick="alert('Hello World!') value="Click me!">

자바스크립트 '출력'의 여러 형태

도큐먼트(HTML)에 내용 출력
	기본 출력		document.write("출력할 내용");
	줄바꿈 출력		document.writeln("출력할 내용");

콘솔창에 로그 출력(디버깅용)
	기본 출력		console.log("출력할 내용");
	구조 출력		console.dir("출력할 내용[자료형]";
	에러 출력		console.err("출력할 내용");

알림창 출력
	알림창		alert("출력할 내용");
