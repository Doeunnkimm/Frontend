메모장 활용
	수업 내용은 메모장을 활용하여 작성한다.

메모장의 글씨 크기 변경 방법
	서식 > 글꼴 > 크기 설정 > 확인
	단축키 : Ctrl + 마우스 위/아래 스크롤

일정 간격 맞추기
	키보드 좌측 상단 Tab 키

작업 디렉토리
	C or D 드라이브 > frontend
	하위디렉토리 : memo, resource, workspace

memo : 수업 내용 기록을 위한 메모장 파일 저장 공간
resource : 주로 다운로드 파일을 저장하기 위한 공간
workspace : 수업 중 작성한 소스(html, css, js 등) 코드 저장 공간

확장자
	.html		HTML파일
	.css		CSS(스타일) 파일
	.scss		확장된 CSS 파일
	.js		자바스크립트 파일
	.jsx		리액트 자바스크립트 파일
	.jpg		투명 색상이 없는 이미지 파일(RGB)	
	.png		투명 색상(배경)이 포함된 이미지 파일(RGBA)
	.mp3		음악 형식의 파일
	.pdf		Acrobat 읽기 전용 문서 파일

확장팩
   	아래는 VSCode에서 Javascript 등을 개발 할 경우 유용한 유틸리티 확장팩 이다.

   	Auto Rename Tag       	Html과 XML 태그의 앞/뒤를 동시에 변경 가능
   	Beautify         			javascript, JSON, CSS, Sass 정렬 
   	Prettier         			코드 포매팅 도구 

   	ESLint            		Lint 설정 기반으로 IDE에 에러 등을 노출 
   	Indent-rainbow         		코드에서 tab과 space들을 구분하기 쉽게 해주는 도구 
   	JavaScript code snippets   	ES6 문법 코드에서 반복되는 코드 조각을 자동 입력
   	Rainbow Brackets      		브라켓(블록)이 여러개인 경우 구분하기 쉽게 해주는 도구 

   	Atom One Dark Theme      	아톰 IDE 기반 색상 VS CODE 다크 테마 

   	Live Server         		HTML을 로컬 호스트(localhost)로 보여주는 가상 서버
