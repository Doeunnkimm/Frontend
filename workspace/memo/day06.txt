CSS(Cascading Style Sheet)
	HTML로 작성된 웹 문서의 스타일의 디자인을 재정의하여
	보다 시각적으로 보여주기 위한 스타일 시트 언어이다.

장점
	웹 구조(HTML)과 디자인(CSS)를 분리하여 관리할 수 있다.
	디자인(CSS)를 변경하여 동일한 HTML을 쉽게 재 디자인할 수 있다.
	반응형 웹을 만들기 위해서는 반드시 필요하다.

스타일 기본 형식
	선택자	{ 속성1: 속성값1; 속셩2: 속성값2; }

선택자(Selector)
	style 받을 HTML 엘리먼트 명, 클래스, 아이디 등을 지정.
선언(Declearation)
	해당 엘리먼트의 속성들을 스타일하고 싶은대로 속성 및 값을 지정
속성(Properties) 프로퍼티
	해당 엘리먼트의 속성을 지정
값(Value)
	해당 엘리먼트의 값을 지정

반드시 중괄호 사이에 속성과 값을 넣어야 한다.
속성과 값은 반드시 하나의 쌍으로 이루어진다.
